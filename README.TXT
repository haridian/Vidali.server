1- Resumen del proyecto.
|=======================
	Vidali surje como una alternativa de código abierto y gratuita a las actuales redes sociales existentes en internet.
	El principal objetivo del proyecto es permitir que cada usuario gestione su propio contenido, tratando de crear un sistema seguro para
	proteger la información privada de cada usuario. Además, al estar instalado en un servidor privado, se evita la dependencia de un dominio
	principal para el funcionamiento de la red (red descentralizada).

2- Instalación.
|=======================
	2.1 Requisitos:
		Plataforma: Cualquiera.
		Servidor web: Apache 2.x, PHP 5.2+, MySQL.
	2.2 Instalacion:
		Para instalar Vidali web server (ya sea de modo local o en un servidor en internet):
		-Colocar la carpeta "Vidali" en la carpeta pública de tu servidor (p.ej: http://localhost/Vidali).
		-Accede a /vidali/install y seguir las instrucciones.
		-¡Listo!, una vez acabes la instalacion elimina la carpeta install por motivos de seguridad.
