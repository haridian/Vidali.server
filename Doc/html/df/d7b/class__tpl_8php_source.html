22  *
23  * --[ fin de página ]------------------------------------------------
24  *
25  * Como pueden apreciar en el corto, pero ilustrativo ejemplo se pueden definir

27  * "--PARTE--" y "--!PARTE--" donde, obviamente, parte será un bloque a llamar, en
28  * caso de nuestro ejemplo, "comentario" para simular un bloque div de comentarios.
29  * También se puede hacer uso de código PHP embebido poniendo antes del bloque "{}"
